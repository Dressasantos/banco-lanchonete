# Banco de Dados – Lanchonete

Projeto acadêmico desenvolvido para a disciplina de Banco de Dados.

## Descrição
Modelagem e implementação de um banco de dados para uma lanchonete, permitindo
o cadastro de clientes, pedidos, entregadores, sanduíches e itens do pedido.

## Tecnologias
- MySQL
- MySQL Workbench

## Estrutura
- Cliente
- Pedido
- Entregador
- Sanduíche
- ItemPedido

## Observação
Projeto com fins acadêmicos.
